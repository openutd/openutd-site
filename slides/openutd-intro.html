What is OpenUTD all about?

Focused on participating in and using open source software, and educating others about open source software.
Usage of Linux as an open operating system.

What is OSS?

Software who's source code has been made publicly available.
Software is distributed with a license that describes how it can be used.

Why?

Benefits:

  • Open Collaboration
  • Security
  • Community
  • Free
  • Freedom and flexibility
  • Quality
